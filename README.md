# 🇮🇳 Indian General Election Result Analysis 2024 | SQL Project

## 📌 Project Overview
This project analyzes the 2024 Indian General Election using SQL Server. The project involves designing a relational database, creating an Entity Relationship Diagram (ERD), importing multiple CSV datasets, and writing advanced SQL queries to generate meaningful election insights.

---

## 🎯 Objectives
- Design a normalized relational database.
- Analyze party performance across India.
- Explore constituency-wise election results.
- Perform state-wise election analysis.
- Extract meaningful insights using SQL.

---

## 🛠 Tools & Technologies
- Microsoft SQL Server
- SQL
- SQL Server Management Studio (SSMS)
- Microsoft Excel
- CSV Files

---

## 📂 Dataset
The project uses five datasets:

- constituencywise_details.csv
- constituencywise_results.csv
- partywise_results.csv
- statewise_results.csv
- states.csv

---

## 🗄 Database Design

- Created a normalized database schema.
- Designed an ERD showing relationships between all tables.
- Implemented Primary Keys and Foreign Keys.
